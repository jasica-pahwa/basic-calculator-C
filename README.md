# basic-calculator-C
A beginner-friendly calculator project built using C programming language.

This project performs basic arithemic operations:
-Addition
-Subtraction
-Multiplication
-Division

##Concepts Used
-Input with 'scanf()'
-'switch' statements
- Conditional Logic
- Error handling for division by Zero

##Purpose
Built as part of my C programming practice to strengthen problem-solving and logic-building skills.
